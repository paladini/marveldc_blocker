Privacy Policy - marveldc_blocker

This privacy policy describes how the marveldc_blocker extension handles user data. The purpose of the extension is to filter and hide Marvel and DC Comics-related content from web pages.

Summary

  • We do not collect personal data.
  • We do not sell, rent, or share user data with third parties.
  • We do not use data for advertising, profiling, credit, or lending.
  • We do not have backend infrastructure to receive user data.

Data Collection

The extension does not collect or store personally identifiable information, financial data, authentication credentials, personal communications, location data, browsing history, or other sensitive data.

Host Permissions and Page Content

The extension uses content scripts to analyze and modify page content locally in the browser (filtering and hiding specific posts or elements). This processing occurs on the user's device and is not transmitted to external servers.

Remote Code

We do not use remote executable code. The extension code is packaged with the published item.

Retention and Sharing

Since we do not collect user data, there is no data retention or sharing.

Policy Changes

This policy may be updated to reflect technical or regulatory changes. When relevant updates occur, this page will be updated accordingly.
